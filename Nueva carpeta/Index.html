El SILEIT es una aplicacion para el control de equipos en instalaciones pertenecientes a las Inmobiliarias del país. El SILEIT presenta varias consultas útiles para el control de estos equipos, no obstante existen algunas variantes que no contempla, como agrupar o filtrar algunos datos dentro del SILEIT, por lo tanto se realizó una aplicación CONSULTAS SILEIT 1.0 donde permite agrupar y mostrar através de una consulta los campos EQUIPOS, SISTEMA DE MANTENIMIENTO, INSTALACION, FECHA Y VALOR, permitiendo buscar y filtrar por cualquier campo, incluso, pueden estar combinados por 2 o más criterios.

Para implementación de CONSULTAS SILEIT se usó EMBARCADERO XE2 y Microsoft SQL Server.
